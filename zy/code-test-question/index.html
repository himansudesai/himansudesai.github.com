This app has an Express server and an Angular frontend. It uses the iTunes music API. You will be editing only the index.html and the albumsCtrl.js.

  1. Use the iTunes api endpoint provided in the albumsCtrl.js file to retrieve a list of all album names by The Beatles.
  2. Display the albums with artwork on the page.
  3. Display the albums sorted by release date.
  4. Display 3 rows of results.
